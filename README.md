# Triode Demand Forecasting

A demand forecasting project using time series analysis and machine learning techniques.

## Getting Started

### Clone the Repository

```bash
git clone https://github.com/v0lab/triode-demand-forecasting.git
cd triode-demand-forecasting
```

### Setup Environment

This project uses [uv](https://github.com/astral-sh/uv) for dependency management. To create the virtual environment and install dependencies:

```bash
uv sync
```

### Main Notebook

The primary analysis and forecasting models are in:

- `notebook/demand_forecasting.ipynb`

## Project Structure

```
├── data/               # Data files
├── notebook/           # Jupyter notebooks
│   └── demand_forecasting.ipynb
├── main.py            # Main Python script
├── pyproject.toml     # Project dependencies
```
